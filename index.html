What is EzLand.js?

Say YES to Web Components!

Основная задача данной библиотеки: быстрый server-side рендерин Вашей веб-страницы, использующий Web Components. Без кучи node-modules, и вообще без Node.js!

При этом библиотека даёт возможность оптимизировать Вашу веб-страницу так, чтобы подгружались только те компоненты, которые действительно нужно показывать в данный момент.
